    Intro

    To use our api, all you need is a token. You’ll be able to use up to 100 calls a month for free. Once you have a valid credit-card you can use our API for unlimited calls. You pay per call. In case you use PDF file with more than one page, each page will count as one call.
    When you call our service, you can either upload a file from your local machine or specify a url. We will detect the text in the file and return you a response with the text inside. All calls are secure using ssl and we do not save any file you use to upload.

    Basic example

    curl --data "url=https://i.imgur.com/fYY6P4Y.png" http://api.scanr.xyz/ocr?token=YOUR_TOKEN&lang=eng

    npm install scanr;
var scanr = require('scanr')('YOUR_TOKEN', {lang: 'eng'});
scanr.ocr('https://i.imgur.com/fYY6P4Y.png', function(err, text){
  console.log(text);
}

//{text: 'hello world!'}

    gem install scanr
scanr = Scanr.new 'YOUR_TOKEN', 'eng'
p scanr.ocr 'https://i.imgur.com/fYY6P4Y.png' 
#{text: 'hello world!'}

    Base HTTP Endpoint

    https://api.scanr.xyz/ocr?token=YOUR_TOKEN

    Language

    You can select any language from this list. The default is English. To use different language pass the parameter lang to your call. For instance: https://api.scanr.xyz/ocr?token=YOUR_TOKEN&lang=heb

    Installation

    To install you can use any of our open-source SDKs. Currently available: Ruby and javascript.

    # No need.

    npm install scanr

    gem install scanr

    Authentication

    In order to authenticate yourself all you need is your token.
    If you use one of the SDKs you can pass a second parameter to select your language.
    The second object is optional and use it if you want to set a specific language (default is English).

    # No need.

    var scanr = require('scanr')('YOUR_TOKEN', {lang: 'eng'})

    scanr = Scanr.new 'YOUR_TOKEN', 'eng'

    OCR

    You can either upload an image or provide its url.

    Upload file

    Use this option if you want to upload a file from your local machine.

    HTTP Request

    POST https://api.scanr.xyz/ocr?token=YOUR_TOKEN

    Query Parameters

    Parameter Description Via
    file The file you want to upload Multipart
    lang The language to use. English is default. Optional. url param

    Response

    {"text": "some text in the image"}

    You can upload any of the supported types. File size can be up to 5MB. 
    curl -F "image=@/path/to/image.png" https://api.scanr.xyz/ocr?token=YOUR_TOKEN

    scanr.ocr('/path/to/image.png', function(err, text){
  //text contains the text in the file
}

    scanr.ocr '/path/to/image.png'

    Existing file URL

    Use this option if your file is publicly accessed over the web.

    HTTP Request

    POST https://api.scanr.xyz/ocr?token=YOUR_TOKEN

    Query Parameters

    Parameter Description Example
    url The url of the file https://i.imgur.com/fYY6P4Y.png
    lang The language to use. English is default. Optional. heb

    Response

    {"text": "some text in the image"}

    The url path have to end with a valid type (`png`, `jpg` etc…) 
    scanr.ocr 'https://path.com/to/image.png'

    scanr.ocr('https://path.com/to/image.png', function(err, text){
  //text contains the text in the file
}

    PDF

    In case you upload a pdf or send it’s url, and the pdf has more than one page thee response will be: {"text": ["text in first page", "text in second page"] }

    Errors

    Usually if you encounter an errr, the response will liik like: {error: ‘error text’}. The scanR API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
